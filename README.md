# ToDo

## Description

Simple web application for handling tasks

## Technologies:

**Backend:**
* **Language:** Python
* **Framework:** Django
* **DataBase:** PostgreSQL

**Frontend:**
* **Framework:** Angular
* **Styles:** CSS
* **State handling:** RxJS

## Features
- Custom registration and account system(password are not hashed)
- Handling tasks: adding/deleting
- Each user has it's own tasks which are saved in database
